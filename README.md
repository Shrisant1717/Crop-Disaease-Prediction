# 🌿 Crop Disease Prediction

A Deep Learning project that detects crop diseases from plant leaf images using **TensorFlow** and **Convolutional Neural Networks (CNNs)**. The model classifies leaf images into **38 different disease categories** to help in early disease detection.

## 🚀 Features

* Detects crop diseases from leaf images
* CNN model built using TensorFlow/Keras
* Supports 38 disease classes
* Training and validation accuracy visualization
* Confusion matrix and classification report
* Saves the trained model for future predictions

## 🛠️ Technologies Used

* Python
* TensorFlow
* Keras
* NumPy
* Pandas
* Matplotlib
* Seaborn
* Scikit-learn



## ⚙️ Installation

Clone the repository:

```bash
git clone https://github.com/yourusername/Crop-Disease-Prediction.git
```

Install the required packages:

```bash
pip install -r requirements.txt
```

Run the training notebook:

```bash
jupyter notebook Train.ipynb
```
